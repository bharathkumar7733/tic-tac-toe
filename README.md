# Classic Tic Tac Toe ❌⭕

A classic browser-based Tic Tac Toe game designed for two players sharing the same screen. The game tracks turns and announces the winner dynamically.

## 🌟 Features
- **Dynamic 3x3 Grid**: Standard interactive grid layout.
- **Turn Switcher**: Automatically swaps between player `X` and player `O`.
- **Winning Combination Scanner**: Instantly identifies win states based on horizontal, vertical, and diagonal lines.
- **Reset Button**: Quickly clears the board and score states to start a fresh game.

## 🛠️ Tech Stack
- **HTML5**: Minimalist canvas grid structure.
- **CSS3**: Layout design with grid styling, transition animations, and visibility triggers.
- **JavaScript**: Click event listeners, multi-dimensional array checks for winning lines, and board state reset logic.

## 🚀 How to Run
1. **Clone the repository**:
   ```bash
   git clone https://github.com/bharathkumar7733/tic-tac-toe.git
   ```
2. Open `index.html` in any web browser and take turns playing!
